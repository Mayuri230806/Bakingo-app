 🍰 Bakingo app – Frontend Web Application

A responsive and visually appealing bakery web application** inspired by the Bakingo platform.
This project is developed using HTML, CSS, and JavaScript, focusing on clean UI design and interactive user experience.

 Overview

The Bakingo app is a frontend-based web application that allows users to browse cakes, view product details, and interact with a basic shopping interface.
It demonstrates strong fundamentals of web design, responsiveness, and DOM manipulation.

 ✨ Key Features

* 🎂 Interactive homepage with featured products
* 📋 Product listing with images and descriptions
* 🛒 Add-to-cart functionality using JavaScript
* 📱 Fully responsive design for mobile and desktop
* 🎨 Clean and modern UI with structured layout
* ⚡ Smooth user experience with basic interactivity


 Tech Stack

 Technology |   Purpose                            
---------- | ---------------------------------- 
 HTML5      | Structure of web pages             
 CSS3       | Styling and responsiveness         
 JavaScript | Interactivity and dynamic behavior 



 Learning Outcomes

* Improved understanding of responsive web design
* Hands-on experience with DOM manipulation in JavaScript
* Better structuring of real-world frontend projects
* Practice of clean and maintainable code organization
